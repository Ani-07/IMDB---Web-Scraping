# IMDB - Web Scraping
Writing a script to scrape IMDB for user ratings

Key Issue – IMDB does not provide a user wise ratings page. A database with movies and comparative rankings can be used to be build recommendation engines based on user ratings

Step 1 – Find list of users

Step 2 – Go to user page and extract ratings given by user for all movies

Step 3 – Add the ratings to an existing list of movies
